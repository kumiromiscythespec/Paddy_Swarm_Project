# Paddy Swarm Work Platform Chassis CAD Pack v0.3

田植え直撒き・腹下除草・高刈りアームに対応するため、亀の体に似せることよりも、作業モジュールを載せるための四輪作業プラットフォームとして再設計したCAD/STLパックです。

## 設計目的

- 中央 Safety Path 運用を前提にした四輪低重心ローバー
- 前方に除草・直播・田植え系アタッチメントを接続
- 腹下に除草撹拌ユニットを搭載
- 後方に高刈り・収穫補助アームを接続
- 上面は亀型マスコット優先ではなく、低い防水・防泥カバーとして整理
- Charge Scute / ソーラー対応面は残すが、作業機能を優先

## 注意

このパックは Grade 0 / R00 の概念・寸法確認用です。実田んぼ投入、電装搭載、水没試験には使わないでください。FDM印刷品は完全防水ではありません。
