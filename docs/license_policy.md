# Paddy Swarm Project ライセンス方針メモ

この文書は、Paddy Swarm Project Grade 0の暫定ライセンス方針です。

## 現時点の方針

Paddy Swarm Projectは、設計情報をできるだけ開き、農家・研究者・学生・個人開発者が試せるようにすることを目指します。

ただし、現時点では正式なライセンスを確定していません。

## 検討中

- ハードウェア設計：CERN Open Hardware Licence系など
- ソフトウェア：MIT / GPL / AGPL など
- 文書：CC BY-SA系
- ロゴ・名称：別管理

## 基本思想

```text
設計は開く。
ただし、Paddy Swarm公式名・検証済み表示・安全基準は守る。
```

誰でも作れるが、粗悪品を「Paddy Swarm公式」と名乗れないようにする方針です。

## 注意

ライセンスが正式に確定するまでは、再配布・商用利用・派生物公開の扱いについてはIssueで相談してください。
