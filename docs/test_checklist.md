# Paddy Swarm Grade 0 試験チェックリスト

この文書は、Paddy Swarm Project Grade 0で公開する3Dプリント部品・模型・試験片を、安全に印刷し、確認し、結果を共有するためのチェックリストです。

Grade 0の目的は、完成ローバーをいきなり実田んぼへ投入することではありません。  
目的は、接続方式、フロート、重心、防水BOX配置、ギアボックス模型、腹下モジュール交換構造などを、小さく安全に検証することです。

---

## 0. 最重要ルール

```text
実田んぼは収入源であり、失敗してよい実験場ではありません。
```

Grade 0では、実田んぼ投入を標準試験にしません。

試験順序：

```text
机上確認
↓
乾いた地面
↓
水槽
↓
柔らかい土
↓
浅泥
↓
水あり模擬環境
↓
小区画・非本番環境
↓
実田んぼ端部
```

---

## 1. 試験レベル

| レベル | 名称 | 内容 |
|---|---|---|
| T0 | 机上確認 | 印刷・組み立て・寸法確認のみ |
| T1 | 乾地確認 | 乾いた床・机・地面での動作確認 |
| T2 | 水槽確認 | 電装なしで浮力・傾斜・浸水確認 |
| T3 | 柔土確認 | 柔らかい土で沈み込みや抵抗を見る |
| T4 | 浅泥確認 | 浅い泥で足回り・フロートの抵抗を見る |
| T5 | 水あり模擬環境 | 模擬田んぼで水・泥・傾斜を確認 |
| T6 | 小区画確認 | 非本番・小区画で人間監視下の確認 |
| T7 | 実田んぼ端部 | 収量影響が少ない端部で限定確認 |

Grade 0の標準目標は **T0〜T2** です。

## 2. 試験前共通チェック

```text
[ ] 試験する部品名とバージョンを確認した
[ ] 使用材料を記録した
[ ] 印刷条件を記録した
[ ] 部品に割れ・欠け・層割れがない
[ ] 接続部が極端にきつすぎない
[ ] 接続部が緩すぎない
[ ] 無理に力を入れない
[ ] 写真を撮る準備がある
[ ] 失敗した場合に記録する準備がある
[ ] 周囲に人・動物・水濡れして困る物がない
[ ] 実田んぼではなく安全な場所で試験する
```

水を使う試験：

```text
[ ] 電装部品を載せていない
[ ] バッテリーを載せていない
[ ] 防水BOX位置には紙タオルを入れた
[ ] 部品が沈んでも回収できる
[ ] 床が濡れても滑らないようにした
[ ] 試験後に乾燥できる場所がある
```

## 4. 接続テストピース確認

対象例：

- `PSR_CONN_001_snap_fit_test_v0_1.stl`
- `PSR_CONN_002_slide_rail_test_v0_1.stl`
- `PSR_CONN_003_dovetail_test_v0_1.stl`
- `PSR_CONN_004_printed_pin_test_v0_1.stl`
- `PSR_CONN_005_wedge_lock_test_v0_1.stl`
- `PSR_CONN_ALL_TEST_PLATE_v0_1.stl`

### 4.1 スナップフィット

```text
[ ] 指で組める
[ ] 過度な力が不要
[ ] 組んだ後に簡単には外れない
[ ] 3回抜き差ししても折れない
[ ] 10回抜き差ししても白化が少ない
[ ] 爪の根元に割れがない
[ ] PETGでも組める
[ ] PLAとPETGで硬さの違いを記録した
```

評価：

```text
組みやすさ：1 / 2 / 3 / 4 / 5
外しやすさ：1 / 2 / 3 / 4 / 5
保持力：1 / 2 / 3 / 4 / 5
破損リスク：低 / 中 / 高
コメント：
```

### 4.2 スライドレール

```text
[ ] 最後まで差し込める
[ ] 途中で引っかからない
[ ] ガタが大きすぎない
[ ] 泥や砂が入った時の余裕を想像できる
[ ] 横方向に抜けにくい
[ ] 指で外せる
[ ] 水洗いしやすそう
```

## 5. サイドフロート断面チェック

```text
[ ] 密閉空気室の形が分かる
[ ] 発泡材挿入部がある
[ ] 区画壁がある
[ ] シャーシ接続レールがある
[ ] 充電窓逃がし形状がある
[ ] 水面ラインを書き込める面がある
[ ] 点検口または確認しやすい形状がある
```

浮力簡易確認：

```text
[ ] 水に浮く
[ ] 片側に極端に傾かない
[ ] 空気室に水が入りにくい
[ ] 水に沈めた後、内部に水が残らないか確認した
[ ] 発泡材を入れた場合の差を記録した
[ ] PETGで印刷した場合の浸水傾向を記録した
```

## 6. 防水BOX配置・傾斜浸水チェック

準備：

```text
[ ] 電装部品は載せていない
[ ] バッテリーは載せていない
[ ] 防水BOX位置に紙タオルを入れた
[ ] 実機想定位置に重しを載せた
[ ] 重しは外れないように固定した
[ ] 水面ラインを記録する準備がある
```

姿勢確認：

```text
[ ] 通常姿勢で紙タオルが濡れない
[ ] 前上がり10度で紙タオルが濡れない
[ ] 前下がり10度で紙タオルが濡れない
[ ] 左傾き10度で紙タオルが濡れない
[ ] 右傾き10度で紙タオルが濡れない
[ ] 前後左右15度で危険箇所を確認した
[ ] 軽く揺らしても防水BOX位置へ水が来ない
[ ] 片側荷重で危険箇所を確認した
```

判定：

```text
[ ] 合格：紙タオルが濡れず、水面ラインにも余裕がある
[ ] 要修正：紙タオルは濡れないが水面ラインが近すぎる
[ ] 不合格：紙タオルが濡れた、または防水BOX位置に水が来た
```

## 9. 推奨する初回試験セット

```text
1. PSR_CONN_ALL_TEST_PLATE_v0_1.stl をPLAで印刷

```

## 10. 最後に

Grade 0で一番大事なのは、完成品を急ぐことではありません。

```text
小さく印刷する。
机の上で壊す。
水槽で沈める。
失敗を記録する。
田んぼで失敗しない。
```

安全に、小さく、現実的に試験してください。
