# Paddy Swarm Work Platform Chassis v0.3

## 変更理由

前回の標準亀甲羅ローバー案は、ブランド性・水切り・防泥ガードとして有効でした。  
しかし、実際の目的は「亀に似た車体」を作ることではなく、以下の作業に対応できる低コストな水田作業プラットフォームを作ることです。

- 田植え・直播
- 腹下除草
- 高刈り収穫補助
- 前進作業、後退帰還
- Safety Path から左右作業レーンへ入る運用

そのため v0.3 では、外観よりもアタッチメント接続・腹下空間・低重心・四輪レイアウトを優先しました。

## 基本構成

| 部品 | 目的 |
|---|---|
| PSR-WP-001 | 四輪作業シャーシ本体。腹下モジュールレール、前後マウント、低重心トレーを含む |
| PSR-WP-002 | 前方クイックアタッチメントマウント。除草・直播・田植え系を想定 |
| PSR-WP-003 | 後方マウント。高刈り・収穫補助・後部処理系を想定 |
| PSR-WP-004 | 腹下除草モジュールのダミー。撹拌軸・タインのクリアランス確認用 |
| PSR-WP-005 | 直播・田植えモジュールダミー。ホッパー・繰り出し・浅溝オープナーを想定 |
| PSR-WP-006 | 高刈り収穫アームダミー。刈刃・穂ガイド・受けトレーを想定 |
| PSR-WP-007 | 低い機能優先カバー。水切り、Charge Scute、ソーラー対応面を含む |

## 重要な設計方針

### 四輪化

1輪を各コーナーに配置します。  
差動制御・左右駆動・将来の四輪独立制御を検討できますが、R00ではまず見た目と配置の確認用です。

### 前方マウント

前方には以下の作業を想定したクイックマウントを置きます。

- 除草用前方ガイド
- 直播ユニット
- 小苗田植え/補植ユニット
- 浅溝オープナー
- 種子ホッパー

### 腹下除草

腹下には除草撹拌ユニットを通せる中央クリアランスを残します。  
R00では撹拌軸ダミーとタインダミーを入れ、地面・泥面との干渉確認を目的にします。

### 後方高刈りアーム

後方には高刈り・収穫補助のためのヒンジ耳とマウントを置きます。  
初期は穂先を切る・受ける・後方処理系に渡すための空間検討用です。

### 低い保護カバー

亀の体に似せるのではなく、低い水切りカバーとして再定義しています。  
上面には以下を残します。

- Charge Scute 予定地
- ソーラー後付け/ペロブスカイト塗布予定面
- 左右水切りリップ

ただし、カバーは第一防衛層です。内部防水カセットはまだ必要です。

## 試験順序

1. PSR-WP-001 をPLAで印刷し、反り・サイズ感・腹下クリアランスを確認
2. PSR-WP-002 / 003 を印刷し、前後マウントの位置を見る
3. PSR-WP-004 を腹下へ仮置きし、除草タインの地面干渉を見る
4. PSR-WP-005 を前方へ仮置きし、重心変化を見る
5. PSR-WP-006 を後方へ仮置きし、後方アームで転倒しないか確認
6. PSR-WP-007 を載せ、水切り・アクセス性・充電窓位置を見る
7. PSR-ASM-002 は印刷せず、全体把握・レンダリング用に使う

## 現場上の注意

- この形はSafety Path運用を前提にする
- 条間に入って作業し、基本は後退帰還する
- 180度その場旋回は標準禁止
- 腹下除草ユニット使用中の旋回は避ける
- 高刈りアームは重心を後ろに動かすので、必ずダミー荷重で確認する
- 直播ホッパーは種子量により重心が変わる
