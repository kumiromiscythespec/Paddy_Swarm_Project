# Paddy Swarm Project Grade 0

田んぼロボットを、まずは無料公開の設計図と試験ログから始めるための公開リポジトリです。

Paddy Swarm Project / 田んぼロボット群プロジェクトは、小規模・家族経営の稲作現場で、人間の負担が大きい作業を小型ロボット群で少しずつ減らすことを目指す試作プロジェクトです。

現段階の目的は、完成品ロボットの販売ではありません。  
まずは **Grade 0** として、3Dプリント用データ、組み立て説明、試験手順、安全ガイド、失敗ログを公開し、興味を持った人が小さく安全に試せる状態を作ります。

---

## 重要：まだ実用機ではありません

このリポジトリのデータは、実験・研究・教育・試作用です。

- 実田んぼで安全に動く保証はありません
- 防水性能は未検証です
- 3Dプリント部品を完全防水容器として信用しないでください
- モーター、バッテリー、制御基板を水に入れる試験は危険です
- 実田んぼは収入源であり、失敗してよい実験場ではありません
- 稲を傷つける可能性のある試験は、必ず小区画・非本番環境から始めてください

推奨する試験順序：

```text
机上
↓
乾いた地面
↓
水槽
↓
柔らかい土
↓
浅泥
↓
水あり模擬環境
↓
小区画・非本番環境
↓
実田んぼ端部
```

## 最初の試験対象

### 1. 接続テストピース

ネジをなるべく使わず、プラモデルのように組み立てるための接続方式を試します。

- スナップフィット
- スライドレール
- 蟻継ぎ
- 印刷ピン
- くさびロック

### 2. サイドフロート断面

水田ローバーが泥や浅水で沈まないための、フロート構造を検討します。

- 密閉空気室
- 発泡材挿入部
- 区画構造
- シャーシ接続レール
- 充電窓逃がし形状

### 3. ギアボックス内部再現モデル

プレゼンや説明に使えるように、左右2モーター、減速ギア、PTO切替、車輪出力、作業ヘッド出力を見える化します。

### 4. 浮力・重心試験用ダミー

電装なしの模型に重しを載せ、水槽で浮力・傾斜・浸水を確認します。

## 将来の拡張構想

### Direct Seeding Arm

育苗工程を減らすための直播アームです。

想定構成：

- 種子ホッパー
- 定量繰り出しローラー
- 浅溝オープナー
- 泥寄せブレード
- 軽鎮圧ローラー

### Weeding Arm

条間中央の初期雑草を処理する除草補助アームです。稲株に近い危険区域は Human Weed Zone として記録し、人間作業に残します。

### High-Cut Harvest Arm

50cm程度の高刈りを標準とする収穫補助アームです。1期目の穂を確保しつつ、茎・葉・節を残し、暖かい年のひこばえ・二番穂を観察します。

### Water Gate

水管理ゲートは高級オプションとして扱います。Grade 0および1田スターターでは、水管理は人間が行います。

## 協力してほしいこと

- STLを印刷してみる
- PLAとPETGで公差を比較する
- スナップ爪の折れやすさを確認する
- スライドレールのきつさを確認する
- フロート断面の印刷しやすさを確認する
- 水槽で浮力を見る
- 重しを載せて傾き方を見る
- 防水BOX位置の問題を見つける
- ギアボックス模型の説明しやすさを評価する
- 組み立て説明書の分かりにくい点を指摘する
- 失敗ログを共有する

失敗ログは非常に重要です。
